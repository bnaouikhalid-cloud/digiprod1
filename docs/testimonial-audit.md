## Approval workflow

Before removing a label, obtain:

1. Written permission to use the person’s name, role, company and quote.
2. Confirmation that wording is still accurate.
3. Separate evidence and approval for any numerical outcome.
4. Approved logo or portrait files with ownership/licence confirmation if visual identity is added.
